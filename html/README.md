# Legacy Hoi An Website

This project is a website for Legacy Hoi An, showcasing its offerings, including rooms, dining options, and more. Below are the details regarding the structure and functionality of the website.

## Project Structure

```
legacy-hoian-website
├── assets
│   ├── css
│   │   ├── style.css          # Main styles for layout, typography, and colors
│   │   └── responsive.css     # Responsive styles for various screen sizes
│   ├── js
│   │   ├── main.js            # Main JavaScript functionality for user interactions
│   │   └── map.js             # JavaScript for map interactions
│   └── images
│       └── .gitkeep           # Keeps the images directory in version control
├── pages
│   ├── index.html             # Homepage with main content
│   ├── about.html             # About page detailing history and services
│   ├── rooms.html             # Rooms page showcasing options and pricing
│   ├── dining.html            # Dining options and menus
│   ├── gallery.html           # Gallery of images
│   └── contact.html           # Contact information and form
├── components
│   ├── header.html            # Header structure with navigation
│   ├── footer.html            # Footer structure with copyright and links
│   └── booking-form.html      # Booking form for reservations
└── README.md                  # Project documentation
```

## Features

- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience across devices.
- **Interactive Elements**: JavaScript is used to enhance user interaction, including dynamic content updates and map functionalities.
- **Comprehensive Information**: Each page provides detailed information about the services offered, including rooms, dining, and contact options.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser to view the homepage.
4. Use a local server for full functionality, especially for JavaScript interactions.
